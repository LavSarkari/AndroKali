# AndroKali
The tool used to run Kali Linux OS in Android Using VNC viewer Application. [No root Requried...]
# Installation

```
pkg install git -y
```
```
git clone https://github.com/LavSarkari/AndroKali
```
```
cd AndroKali
```
```
bash AndroKali.sh
```
